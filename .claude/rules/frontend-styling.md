# Frontend & Styling Guidelines

- All new HTML/CSS should use Bootstrap and regular CSS classes
- All custom CSS goes in `web/static/web/styling.css`
- As views are modified, craft new Bootstrap-compatible code
- Let Tailwind and Bootstrap co-exist during the migration
- Users will primarily access pages on mobile devices — design mobile-first
- Make all pages responsive
