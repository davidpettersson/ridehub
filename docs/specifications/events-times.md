# Events: Times

## Start time

 - Can be any time, past or future
 - Must always be set

## End time

 - Always greater than or equal to start time
 - Optionally set
 - Offers quick set durations:
   - Add 1 hour
   - Add 1.5 hours
   - Add 2 hours
   - Add 3 hours
   - Add 4 hours
   - Add 5 hours
   - Add 8 hours

## Registration close time

 - Always less than or equal to start time
 - May be blank if there exists an external registration URL
 - Must be set if no external registration URL
 - Offers quick set options:
   - At start
   - 1 hour before start
   - 2 hours before start
   - 3 hours before start
   - Same day at noon
   - Day before at noon
   - Day before at 4 pm
   - Day before at 5 pm
   - Day before at 8 pm
   - Day before at 9 pm
